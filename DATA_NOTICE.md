# Example data notice

`win001.h5` is a deidentified waveform window included solely to demonstrate
and test Waveform. It was derived from a third-party research recording used in
the author's research project. Source record names and identifying metadata
were removed when the HDF5 container was rebuilt. The public file retains only
the waveform, generic signal metadata, channel labels, and sampling rates
needed to exercise the application.
