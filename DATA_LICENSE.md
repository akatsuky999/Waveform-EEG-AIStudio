# Example data

`win001.h5` is a deidentified waveform window included for demonstrating and
testing EEGViewer. Source record names and identifying metadata have been
removed by rebuilding the HDF5 container. The public file retains only the
waveform, generic signal metadata, channel labels, and sampling rates needed to
exercise the application.
